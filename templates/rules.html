LOG



Rule Name can be Anything.
In the input field, you can directly assign values to your variables.
Some variables are predefined. Eg: baserate, creditscore, interest, income, employment.

Rules like multiple direct assignment and conditional statement (and) are accepted.
Rule Format:
Direct Assignment: (variable) = (value)
Conditions: if (variable (=, >, <=, !=, <, >=) value) then (variable = value)

Example:
baserate = 12.7
interest = 12.7
if creditscore > 890 and income >= 500000 then interest = 10.2
